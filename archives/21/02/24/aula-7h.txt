Regras para definição de variáveis:
    Mantenha os nomes significativos, não seja vago para facilitar a leitura do código
    Nomeie apenas com nomes, números e _
    O nome deve começar com um caractere ou _
    Sem espaços.
    Sem caracteres especias &, %, $, etc...
    Não utilize palavras reservadas da linguagem (Em C por exemplo: float, int, class, void, etc...)

Tipos de dados: 
    Integer (int): números inteiros apenas
    float (float): números reais, decimais etc...
    string (string): Texto, palavras. ==>    int 5 É DIFERENTE DE  string '5'
